An IPv6 address consists of eight groups of up to 4 characters consisting of the digits 0-9 and letters a-f.


2001:db8:44:e:12d:ff00:42:8329



Omitting leading zeros

If a group starts with one or more zeros, the zeros can be left out.


0042 → 42


Zeros appearing after a non-zero character must be kept in.


ff00 → ff ❌


4002 → 42 ❌



Compressing multiple groups of zeros

If several groups in a row contain all zeros, they can be compressed into a single :: to represent where the excluded groups begin and end.

5fa3:d43:0:0:0:a00a:54:4329 → 45fa3:d43::a00a:54:4329


However, this trick can only be used once in your address


55:5ff3:0:0:0:49:0:0 → 55:5ff3::49:: ❌


55:5ff3:0:0:0:49:0:0 → 55:5ff3::49:0:0 ✅


55:5ff3:0:0:0:49:0:0 → 55:5ff3:0:0:0:49:: ✅



Encapsulating an IPv4 Address

The last two groups of an IPv6 address can be replaced with an IPv4 address.

ffff::192.168.1.1 ✅
